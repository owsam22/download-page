🛠️ Install Instructions

  1. Download and unzip the extension
  2. Open chrome://extensions
  3. Enable Developer Mode
  4. Click Load Unpacked and select folder

✨ Features

  • Per-tab volume control
  • Mute/unmute buttons
  • Light/dark mode toggle
